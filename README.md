# Spotify UI Clone 

This project is a simple **Spotify Web Player UI clone** built using **HTML & CSS**.  
It replicates Spotifyâ€™s layout including:

- Sidebar navigation  
- Library section  
- Main content area with cards  
- Music player at the bottom  
- Responsive design for smaller screens  

This project is great for beginners to practice **CSS Flexbox, layout structuring, and UI design**.

# Technologies Used
- HTML5  
- CSS3  
- Google Fonts  
- Font Awesome Icons  

# Features
- Clean and modern UI  
- Sticky navigation bar  
- Fully responsive sidebar  
- Card layout for songs/playlists  
- Bottom music player with controls  

# How to Use
1. Download or clone the repository  
2. Keep all images in the **same folder** as the HTML file  
3. Open the `index.html` (or your HTML filename) in any browser  


# Note
- This is only a **UI clone**. Music playback is **not functional**.
